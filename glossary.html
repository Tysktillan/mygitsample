I bilden till höger illustreras vägen från att eleven skriver in ett svar till att systemet har gått igenom samtliga regler.
Det finns två typer av feedback - positiv och negativ feedback. De båda aktiveras vid olika förutsättningar. Generellt kan man säga att negativ feedback har en möjlighet att aktiveras när eleven svarar fel. Positiv feedback aktiveras när eleven svarar rätt, men har brutit några regler. Läs mer om de specifika feedbacksen via menyn till vänster.
  • 1. Eleven svarar - Om ogiltiga tecken används blir det syntaxfel
  • 2a. Evaluation type - Exempelvis om evaluation type 'eval eq numeric' i fliken solutions används måste elev svara med en ekvation, t.ex 'x = 2'. Om elev svarar endast '2' går det till standard feedback och rättas som fel.
  • 2b. Kollar regler - Är svaret rätt enligt de olika reglerna för evaluation type, likt exemplet ovan gällande ekvation (angavs t.ex. svar x=2 eller x=3).
  • 3. Positiva feedback regler - Är svar korrekt får eleven rätt svar om inte någon positiv feedback har satts. Läs mer om dessa här
  • 4. Negativa feedback regler- Är svar felaktigt får eleven fel. Beroende på vilket fel svar som angivits kan negativ feedback aktiveras med anpassad text. Läs mer om dessa här
